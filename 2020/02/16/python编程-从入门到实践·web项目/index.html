    创建应用程序

  7. 创建应用程序 在创建应用程序后，怎么让Django值得你创建了一个应用程序，并且将其加入到项目中呢？
    打开位于项目文件夹的setting.py，将你的应用程序名字加到
    INSTALLED_APPS里面。
  8. 定义模型
    如果你想要实现什么功能的话，第一步就是在应用程序文件夹中的models.py中定义一个新的类，并且在类中添加你实现功能所需要的信息什么的。这样你就把一个功能写出来了。 (因为版本更新，这里的代码和书上的部分不相同。)
    功能写好了，功能能够实现，还需要数据库的支持。 每次定义模型后，都要进行同样的操作，称之为数据库迁移。
  9. Django管理网站
    网站管理员可以使用管理网站，普通用户不能。
  10. 1创建超级用户 输入用户名和密码
  11. 2注册模型
    打开应用程序文件夹中的admin.py 你可以访问http://localhost:8000/admin/ 来进入管理网站
